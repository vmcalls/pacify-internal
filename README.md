# Pacify Internal
Simple internal i made for the game called "Pacify" which is a co-op horror survival game. ( THIS IS UNFINISHED AND YOU MIGHT EXPERIENCE LACK OF FEATURES OR CRASHES )

# FEATURES
**ESP**
- 2D Boxes
- Name ESP
- Both work on monsters (from the two first maps) and some entities from the second map.

**2. Exploits**
- Freeze monster.
- Player fly + noclip
- Ghost ignores you
- Speed hack

# Usage
- Generate game SDK and include files into the project
- Build the project
- Inject the DLL into any process (feel free to use any injector)

- minhook.lib
